## Review Notes Template

```
ToneType is a keyboard extension that enhances messages with emojis and
Unicode styling based on detected emotional tone.

TO TEST:
1. Install the app
2. Go to Settings → General → Keyboard → Keyboards → Add New Keyboard → ToneType
3. Tap ToneType and enable "Allow Full Access"
4. Open Messages or Notes app
5. Switch to ToneType keyboard (globe icon)
6. Type any message (e.g., "I'm so excited about this!")
7. Tap the ✨ button to see the enhancement

NOTES:
- The keyboard requires Full Access to make network calls to our AI service
- No data is stored or logged
- For testing AI features, the app works with basic offline detection
- Contact: review@tonetype.app for any questions
```
